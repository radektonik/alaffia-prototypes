# Alaffia Prototypes

Clickable HTML prototypes for the Alaffia claims app.

## Itemized Bill — Main App layout (`index.html`)

A pixel-faithful recreation of the Itemized Bill / claim-detail screen (claim `dc00007`)
from the **Main App — Product design — Alaffia** Figma file.

Self-contained — inline CSS/JS, Geist loaded via Google Fonts. Open `index.html` in a
browser, or enable GitHub Pages to view it online.

**Highlights**
- Fixed 1440px layout matching the Figma frame; main table and findings sub-table share one
  13-column grid for perfect column alignment.
- Dual **Items** / **Findings** select-all checkboxes with mutually-exclusive selection and a
  floating action bar (Create / Accept / Edit findings).
- Expandable per-item findings, the `#` column shows each item's finding count, status pills,
  and Create / Edit Findings modals.
